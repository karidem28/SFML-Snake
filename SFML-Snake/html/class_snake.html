Detailed Description

"Snake Game" - Snake Class.

The Class implements functionality for the snake hero object in game.

Class describes of Snake functionality and properties, as:
Changing direction, size of tail, checking of intersecton with itself.

Example:

snake = new Snake(Vector2f(100,100),0,20);
if (event.type == Event::KeyPressed)
{
if (event.key.code == Keyboard::Left) {
snake->turn_left();
}
if (event.key.code == Keyboard::Right) {
snake->turn_right();
}
}
bool IsIntersected = sn->self_intersection();
if (IsIntersected) DoSomethingToStopGame();
snake->grow(10);
delete(sn);

◆ turn_left()

void Snake::turn_left ( )

Method changes angle to counterclock-wise on Snake::delta value of movement Snake.
On each continuous call of method the Snake::delta increace to +0.5 to make feeling of turning acceleration.
Metod can be called only one times per game tick to avoid unpredicted movement or states.

◆ update()

void Snake::update ( )

Method updates all Snake params on the each Game tick.
It adds new array element with new coords to the begin of array
If the count of sections of Snake Hero in array greater or equal then size of Hero then remove one element from tail's end
Inserting new element with new coords to head and removing one element from end makes movement effect
Metod can be called only one times per game tick to avoid unpredicted movement or states.
