Method updates all \mbox{\hyperlink{class_snake}{Snake}} params on the each \mbox{\hyperlink{class_game}{Game}} tick.~\newline
It adds new array element with new coords to the begin of array~\newline
If the count of sections of \mbox{\hyperlink{class_snake}{Snake}} Hero in array greater or equal then size of Hero then remove one element from tail\textquotesingle{}s end~\newline
Inserting new element with new coords to head and removing one element from end makes movement effect~\newline
Metod can be called only one times per game tick to avoid unpredicted movement or states.
