Example of the snake game with using SFML Library
The main goal of the game is eating the green apples and avoid of walls and self intersection.

Usage keys:
Left arrow key - turn direction counterclock-wise. Continius key pressing accelerates the speed of turn.
Right arrow key - turn direction clockwise. Continius key pressing accelerates the speed of turn.
Up arrow key - restarts the level after the snake die.




File levels.txt contains level's diagaram:
level contains keyword Level#XX at begin - where XX is index number
Then contains list of box coordinated - each line contains one set of wall coodinates.
and then level description ends by '###' string.

Example:

Level#1
0 590 600 10
0 0 10 600
590 0 10 600
0 0 600 10
###
Level#2
0 590 600 10
0 0 10 600
590 0 10 600
0 0 600 10
295 50 10 500
###
Level#3
0 590 600 10
0 0 10 600
590 0 10 600
0 0 600 10
295 50 10 500
50 295 500 10
###
